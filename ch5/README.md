# Coding Factory - Java Exercises

Ασκήσεις από το μάθημα Java του Coding Factory.

## Περιεχόμενα

### Κεφάλαιο 5 - Methods
- **CalculatorApp** - Εφαρμογή υπολογιστή με μενού επιλογών
- **StarsApp** - Εκτύπωση αστεριών σε διάφορα σχήματα  
- **CelsiusToFahrenheit** - Μετατροπή θερμοκρασίας
- **IsPrime** - Έλεγχος αν αριθμός είναι πρώτος
  
## Απαιτήσεις

Java 17 ή νεότερη (λόγω χρήσης switch expressions με `->`).
